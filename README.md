# Python Practice Programs

这个项目收集了 4 个命令行 Python 小程序，分别练习了数据组织、文本处理、文件持久化和安全输入处理。

## 1. 学员花名册管理器
文件：[student_manager.py](./student_manager.py)

这个程序把学员资料保存在内存中的一个字典里，适合练习“按名字查找信息”的基本思路。运行后会显示菜单，用户可以录入学员、按姓名查找、删除记录，也可以一次性列出当前所有学员。

应用上，它像一个简化版的通讯录或报名表管理工具，重点在于练习结构化保存数据，以及对日期格式和空输入做基本校验。

运行方式：

```powershell
python student_manager.py
```

## 2. 文本词频统计器
文件：[text_frequency.py](./text_frequency.py)

这个程序会读取用户输入的一段文本，提取其中的英文单词，并把单个汉字也当作统计单位。统计完成后，它会按照出现次数从高到低输出结果；如果次数相同，再按内容排序。

它可以用来做非常基础的文本分析，比如快速看看一段英文里哪些词最常出现，或者粗略观察一段中英文混合文本里的高频内容。

运行方式：

```powershell
python text_frequency.py
```

## 3. 待办事项清单
文件：[todo_list.py](./todo_list.py)

这个程序提供了一个简单的命令行待办系统。用户可以新增任务、查看所有任务，并按编号把某一项标记为已完成。所有数据会写入同目录下的 `todos.json`，所以关闭程序后再次启动，之前的内容还会保留下来。

它的实际价值在于练习“程序状态落盘”这件事，也就是把内存里的列表保存成文件，再在下次运行时读回来。同时，它也对缺失文件和损坏数据做了保护，避免一启动就报错。

运行方式：

```powershell
python todo_list.py
```

说明：
- `todos.json` 会在首次保存待办事项时自动生成

## 4. 安全的计算器
文件：[safe_calculator.py](./safe_calculator.py)

这个计算器不是分步输入数字和运算符，而是直接接收一整条算式，例如 `1+2/3*5`。它支持 `+ - * /`、括号、正负号，也能把用户输入的 `x` 或 `X` 自动当成乘号处理，末尾带不带 `=` 都可以计算。

它更像一个“安全版表达式求值器”：程序会先解析输入结构，再只允许加减乘除这些合法运算参与计算，因此既能处理除零和非法输入，也不会把任意文本当成代码执行。

运行方式：

```powershell
python safe_calculator.py
```

## 通用运行说明

请先确保本机已经安装 Python 3，然后在当前项目目录打开终端，使用下面的命令运行任意一个程序：

```powershell
python 文件名.py
```
